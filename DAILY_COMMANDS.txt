------------------------------------------------------------
RUFLO BRIDGE — NEW AI FEATURES (2026-04-20)
------------------------------------------------------------
4 new modules in agents/ruflo_bridge/ add persistent AI memory:

1. SEMANTIC TRADE JOURNAL (semantic_journal.py)
   - Indexes every trade with session, regime, R-multiple
   - Natural language search via /query Telegram command
   - Rebuilt every 30 minutes automatically

2. SELF-HEALING AUTOPILOT (autopilot.py)
   - Watches 5 critical agents, auto-restarts on crash
   - Up to 10 restarts per agent with 30s cooldown
   - Telegram alert on crash AND on recovery
   - Check status with /agents command

3. OPTIMIZER MEMORY (optimizer_memory.py)
   - Records every nightly backtest combo + regime
   - Next night: skips known-bad combos, seeds good ones
   - Makes 30-combo budget progressively smarter
   - Check stats with /optmem command

4. WEB NEWS SCRAPER (agents/news_sentinel/web_scraper.py)
   - Scrapes Kitco + FXStreet + Investing.com RSS
   - Zero API keys, zero rate limits
   - News Sentinel AI uses it first (before DuckDuckGo/NewsAPI)
   - Manual check with /webnews command

CLAUDE CODE SESSIONS:
   - Every session auto-receives MIRO live state in context
   - No need to re-explain balance/regime/deployment status
   - State injected via SessionStart hook from session_context.json
